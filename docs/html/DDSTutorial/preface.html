Preface¶

About the DDS Tutorial¶

The DDS Tutorial is included with the OpenSplice DDS Documentation Set. This Guide is the starting point for anyone using, developing or running applications with OpenSplice DDS.

This DDS Tutorial contains:

  • a general introduction to the Data Distibution Service (DDS)
  • descriptions of how to develop applications which exploit features of the DDS
  • etc.

Intended Audience

The DDS Tutorial is intended to be used by anyone who wishes to make use of a DDS product.
