Getting Started

  • Getting Started Guide (pdf) (html) - the document that users should read first. The Getting Started Guide provides:
    • information for running your first example
    • basic installation and configuration information
    • how to license your installation
  • Evaluating and Benchmarking OpenSplice (pdf) (html) - the document that users should read to help get the best out of OpenSplice DDS:
    • choosing your architectural model
    • choosing the right networking service
    • how to benchmark to get the best results
  • OpenSplice Tutorial (pdf) - A step-by-step guide to using OpenSplice DDS APIs. A good guide to understand the concepts of DDS, based on the C API with other language examples available in the examples/dcps/tutorial directory.
  • DDS Tutorial (pdf) (html) - A step-by-step guide to using the DDS API. A good guide to understand the concepts of DDS.
  • Examples (html) - information on how to build and run the DCPS examples.
  • OpenSplice RMI User Guide (pdf) (html) - an introduction to using OpenSplice RMI to achieve request/reply communication over OpenSplice DDS.

API Reference Manuals

  • DCPS ISO/IEC C++ Documentation (html) - a detailed reference describing the OpenSplice DDS DCPS API for programming ISO/IEC C++ including examples. This C++ API is the latest, simpler DDS API.
  • DCPS ISO/IEC C++ 2 Documentation (html) - a detailed reference describing the OpenSplice DDS DCPS API for programming ISO/IEC C++ 2 including examples. This C++ API is a new version of DCPS ISO/IEC C++ API.
  • DCPS Java 5 Documentation (html) - a detailed reference describing the OpenSplice DDS DCPS API for programming Java 5 including examples. This Java API is the latest, simpler DDS API.
  • DCPS Java Reference Guide (pdf) - a detailed reference describing the OpenSplice DDS DCPS API for the Java Language.
  • DCPS C Reference Guide (pdf) - a detailed reference describing the OpenSplice DDS DCPS Application Programmers Interface (API) for the C Language.
  • DCPS C++ Reference Guide (pdf) - a detailed reference describing the OpenSplice DDS DCPS API for the C++ Language.
  • DCPS C# Documentation (html) - a detailed reference describing the OpenSplice DDS DCPS API for the C# Language and other release notes.
  • QosProvider Documentation (All languages) (html) - a detailed reference describing the OpenSplice DDS DCPS API for the QosProvider.
  • OpenSplice Streams API Reference (pdf) OpenSplice Streams API Reference (html) - a detailed reference describing the OpenSplice Streams API that supports a common data-distribution pattern where continuous flows or streams of data have to be transported with minimal overhead and therefore maximal achievable throughput.
  • OpenSplice RMI C++ API Reference (html) - a detailed reference describing the OpenSplice RMI API in C++.
  • OpenSplice RMI JAVA API Reference (html) - a detailed reference describing the OpenSplice RMI API in Java.
  • OpenSplice QoS Provider API Reference (html) - a detailed reference on the OpenSplice QoS Provider API.
  • Dynamic Network Partitions for RTNetworking API Reference (html) - a detailed reference on topic-API for the Dynamic Network Partitions for RTNetworking feature.
  • ISO C++ 2 FACE API Documentation (html) - a detailed reference describing the OpenSplice DDS FACE API for programming ISO/IEC C++ 2 including examples.
  • JAVA FACE API Documentation (html) - a detailed reference describing the OpenSplice DDS FACE API for programming JAVA including examples.
