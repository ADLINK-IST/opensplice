Building your own Corba or C++ API

The OpenSplice DDS DCPS APIs for the C++ and Java language bindings with Corba cohabitation are delivered using a specific compiler and ORBs. The API for standalone C++ is also compiler-specific.

To be able to use a different compiler and/or ORB, the support libraries for these APIs may be rebuilt from the supplied source code, as described in this note.
