SUMMARY

  • HelloWorld

    The basic HelloWorld example is used to illustrate the necessary steps to setup DCPS entities

  • WaitSet

    The basic HelloWorld_Waitset example is used to illustrate the use of waitset and conditions :

    • ReadCondition
    • StatusCondition
    • QueryCondition
    • GuardCondition

  • Listener

    The HelloWorld_Listener example is used to illustrate use of listeners

  • ContentFilteredTopic

    The StockQuote example is used to illustrate message filtering based upon Content-Based subscription.

  • QueryCondition

    The StockQuote example is used to illustrate message filtering through use of Queries.

  • Lifecycle

    The Lifecycle example is used to illustrate the different lifecycle states of the sample and the instance.

  • Durability

    The durability example illustrates the ability of late joining readers to obtain data that has already been published before these readers were created.

  • Ownership

    The ownership example is used to illustrate the concept of Shared Ownership, controlled through QoS, in DDS.

  • BuiltInTopics

    The BuiltInTopics example is used to illustrate the use of built-in topics to obtain and print all nodes running in the domain.

Important Note! (Applies to Windows only)

We have provided a number of start.bat batch files to run the publisher/subscriber of each example. (This is described in the Building/Running examples in Visual Studio 2005/2008 section of each of the examples). We have assumed that the examples are build in a Release configuration. If you build the examples in a Debug configuration, then you will have to ammend the batch files to point to the Debug output directory so they can locate the executable correctly!
